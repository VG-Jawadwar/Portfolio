HydroSense Application

HydroSense is an Android application designed for real-time water quality monitoring using IoT-based sensors. Built with Flutter, it displays readings like TDS, pH, turbidity, and temperature, collected via Arduino and sent to the app through Firebase. The app features dynamic dashboards, historical analysis graphs, and alert notifications when water quality exceeds safe limits. I developed the full mobile interface, integrated Firebase for real-time data syncing, and ensured accurate visualization of sensor data.

JobConnect Application

Job Connect is a Flutter-based Android application designed to simplify the job search process for users. It enables job seekers to browse listings and apply, while employers can post and manage vacancies. The app features secure login, smooth navigation, and a clean, user interface. I managed the entire development process, from UI design to backend integration using APIs and local storage. The project emphasizes a reliable and user-centric platform to effectively connect talent with opportunities.

ClearDues Application

ClearDues is a Java-based desktop application developed to simplify student fee management in academic institutions. It enables administrators to view dues, send email reminders, update student records, and generate payment receipts. The application integrates JavaMail API for email communication and iTextPDF for automated receipt generation. I designed and implemented the complete system, including modules for login, dues tracking, student updates, and payment processing. This project ensures efficient, paperless, and organized fee handling through a user-friendly interface.
